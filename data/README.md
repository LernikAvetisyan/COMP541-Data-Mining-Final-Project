# Data Folder

This project uses the Instacart training-order dataset.

The raw CSV files are not included in this repository. Download the dataset from the course source, Canvas, or Kaggle if permitted, then place the required files in this folder.

Required files:

```text
aisles.csv
departments.csv
products.csv
orders.csv
order_products__train.csv
```

The project does **not** require `order_products__prior.csv`.

Expected structure:

```text
data/
├── aisles.csv
├── departments.csv
├── products.csv
├── orders.csv
└── order_products__train.csv
```

If the notebook uses a different path, update the file paths in `Final_Project.ipynb` before running the analysis.
